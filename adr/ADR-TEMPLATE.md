## Context

What is the situation that forces a decision? What constraints, forces, and prior decisions bear on
it? State the problem so plainly that the decision reads as almost inevitable — or, if it is a
genuine toss-up, say so.

## Decision

The single decision, stated in the active voice: "We will …". Be specific enough that someone can
tell whether an implementation obeys it.

## Consequences

- **Positive:** what gets better.
- **Negative:** what gets worse or harder. (An ADR with none is incomplete.)
- **Neutral / follow-ups:** what this obliges us to do next; which tripwire or test now guards it.

## Compliance

How will we *know* the decision is being honoured? Name the test, tripwire, or review that enforces
it — a decision with no enforcement is a wish.
